# ✋ HAND DIMENSION — Digital Reality Interface
### Developer: Karndeep Baror

---

## 🚀 HOW TO USE

1. Open `index.html` in a browser (Chrome recommended)
2. Click **"GRANT ACCESS & LAUNCH"** to allow camera
3. Show your hand to the camera and use gestures!

---

## 🖐️ GESTURE GUIDE

| Gesture | Action |
|---|---|
| ☝️ One Finger (Index) | **DRAW** — Write/draw in the air |
| ✌️ Peace Sign | **PARTICLE BURST** — Explosion of particles |
| ✊ Fist | **FREEZE** — Pause drawing |
| 🖐️ Open Palm | **UNFREEZE** — Resume + Clear freeze |
| 🤙 Shaka (thumb+pinky) | **NEURAL WEB** — Glowing connection network |
| 👉 Gun (thumb+index) | **RIPPLE BLAST** — Shockwave ripple effect |

## 🎨 FEATURES

- ✅ Real-time hand tracking via MediaPipe
- ✅ Air drawing with glowing trails
- ✅ Particle burst system
- ✅ Neural web / connectivity visualization
- ✅ Ripple shockwave effects
- ✅ Orbiting JARVIS-style icon on palm
- ✅ FPS counter & live stats
- ✅ Multiple colors + tools in side panel
- ✅ Scan-line + grid HUD overlay
- ✅ Boot sequence animation
- ✅ Fully responsive

---

## 🛠️ TOOLS (Side Panel)

- **Draw** — Freehand air drawing
- **Erase** — Switch to erase mode
- **Particles** — Permanent particle mode
- **Neural Web** — Connectivity graph mode
- **Color Dots** — Pick stroke color
- **Clear (trash)** — Wipe canvas
